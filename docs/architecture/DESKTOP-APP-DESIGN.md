## Overview

The desktop application provides an **enterprise-grade graphical interface** with advanced features for professional teams, while the CLI remains free and open source for the developer community.

**Both editions share the same core engine** - the portal/heart of cdev.

---

## Architecture: Shared Core Engine

The **Portal Core** is the heart of cdev - a single, unified engine that powers both Community (CLI) and Enterprise (Desktop) editions. This ensures feature parity, consistent behavior, and easier maintenance.

### Portal Core Components (`internal/`)

| Component | Path | Description |
|-----------|------|-------------|
| **Event Hub** | `internal/hub/` | Central event dispatcher, fan-out to all subscribers |
| **Claude Manager** | `internal/adapters/claude/` | Claude CLI process management, streaming I/O |
| **Git Tracker** | `internal/adapters/git/` | Git operations, status, staging, commits |
| **File Watcher** | `internal/adapters/watcher/` | fsnotify-based file change detection |
| **Session Cache** | `internal/adapters/sessioncache/` | Session parsing and element extraction |
| **HTTP Server** | `internal/server/http/` | REST API endpoints |
| **WebSocket Server** | `internal/server/websocket/` | Real-time event streaming |
| **Domain Events** | `internal/domain/events/` | Event type definitions |
| **RPC Methods** | `internal/rpc/handler/methods/` | JSON-RPC 2.0 method handlers |
| **App Orchestrator** | `internal/app/` | Wires everything together, lifecycle management |

### Entry Points

```go
// Community Edition (CLI)
// cmd/cdev/main.go
func main() {
    app := app.New(config)  // Uses Portal Core
    app.Start(ctx)          // Headless daemon mode
}

// Enterprise Edition (Desktop)
// cmd/cdev-desktop/main.go
func main() {
    app := app.New(config)  // Uses SAME Portal Core

    // Additional: Wails GUI wrapper
    wails.Run(&options.App{
        OnStartup: func(ctx context.Context) {
            app.Start(ctx)  // Start Portal Core
        },
        Bind: []interface{}{
            app,            // Expose to frontend
        },
    })
}
```

## Technology Stack

### Recommended: Wails v2 (Go + Web Frontend)

**Why Wails?**
- **Native Go integration** - Shares code with existing cdev backend
- **Lightweight** - ~10MB binary vs ~150MB+ for Electron
- **Cross-platform** - macOS, Windows, Linux from single codebase
- **Modern frontend** - Use React/Vue/Svelte with TypeScript
- **Native OS integration** - System tray, notifications, file dialogs
- **Single binary** - Easy distribution

## Data Model

### Repository Configuration

```typescript
interface Repository {
  id: string;                    // UUID
  path: string;                  // Absolute path
  displayName: string;           // User-friendly name
  gitRemote?: string;            // Origin URL
  gitBranch?: string;            // Current branch
  lastActive?: Date;             // Last session time
  isIndexed: boolean;            // Whether indexed for search
  watchEnabled: boolean;         // Auto-watch file changes
  gitDiffEnabled: boolean;       // Track git diffs
}

interface AppConfig {
  // Server
  httpPort: number;
  bindAddress: string;

  // Claude
  claudePath: string;
  skipPermissions: boolean;
  sessionMode: 'new' | 'continue' | 'resume';

  // Startup
  launchAtLogin: boolean;
  startMinimized: boolean;
  autoConnectLast: boolean;

  // Appearance
  theme: 'dark' | 'system';

  // Repositories
  repositories: Repository[];
  activeRepositoryId?: string;
}
```

  - macOS: `~/Library/Logs/cdev/`

## Build & Distribution

### Development

```bash
# Install Wails
go install github.com/wailsapp/wails/v2/cmd/wails@latest

# Development mode
wails dev

# Build for current platform
wails build

# Build for specific platform
wails build -platform darwin/universal  # macOS (Intel + Apple Silicon)
wails build -platform windows/amd64     # Windows
```

### Distribution

**macOS:**
- DMG installer with drag-to-Applications
- Notarized and signed
- Optional: Mac App Store

**Windows:**
- MSI installer
- Optional: Microsoft Store
- Portable ZIP version

## Summary

The cdev desktop app provides a **visual interface** for managing repositories and connecting mobile devices, while preserving the terminal experience for power users. Built with **Wails** (Go + React/TypeScript), it shares the backend with the CLI and follows the **Pulse Terminal Design System** from cdev-ios for visual consistency across platforms.

**Key Value Props:**
1. **One-click QR scanning** - Instant mobile connection
2. **Multi-repo management** - Easy switching between projects
3. **Visual status monitoring** - See Claude state at a glance
4. **Cross-platform** - macOS and Windows from single codebase
5. **Lightweight** - ~10MB binary, not Electron-heavy
